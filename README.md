# 👟 Thala 7 Sneaker Store (Sneaker Selling Website)- Task-11

An E-Commerce Website UI **Sneaker Selling Store** built using HTML & CSS, featuring product display, shopping cart, and payment interface.

---

## 🚀 Features

- Header & Navigation
    - Fixed header with logo and navigation menu
    - Search bar with icon
    - User profile and cart icon section
- Hero Section
    - Two-column layout (text + image)
    - Call-to-action button with hover animation
    - Highlighted branding text
- Achievements Section
    - Grid layout displaying stats 
- Product Display
    - Main product preview with image gallery
    - Price section with styled highlight
    - Quantity selector and Add to Cart button
    - Product description and discount info
- Shopping Cart
    - Table-based cart layout
    - Column width control using nth-child
    - Quantity controls (+ / -)
    - Delete item option
- Coupon & Total Section
    - Coupon input and apply button
    - Price breakdown (subtotal, total)
    - Checkout button
- Payment Section
    - Split layout (form + summary)
    - Payment method selection (Card / Other)
    - Card details form (expiry, CVC, country, ZIP)
    - Styled input fields with focus & active states

---
